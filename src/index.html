"SIENTETE SEGURO Y PROTEGE
TU INFORMACION PERSONAL"

Ahora puedes crear contraseñas dificiles
de hackear para tus cuentas personales como
redes sociales, paginas web, correos electronicos, etc.

Crear contraseñas dificiles de descifrar protege tu informacion personal.

Las contraseñas dificiles evita y protege robos informaticos a las cuentas personales.

Tener diferentes contraseñas evitas que puedan hackear todas tus cuentas personales.

Para comenzar a crear tus contraseñas seguras:

Si deseas cifrar una contraseña usando los caractereres de mayusculas y minusculas, dale click en CIFRAR.

Si deseas descifrar una contraseña, dale click en DESCIFRAR.

Escribe una contraseña nueva, indicando un numero de desplazamiento(offset) como clave de recorrido para que se pueda cifrar.

Ahora puedes usar esta contraseña cifrada para tus cuentas personales y asi poder evitar cualquier intento de robo cybernetico.

Escribe la contraseña cifrada, indicando tambien un numero de dezplazamiento(offset) ya utilizado al inicio de crear tus contraseñas.

Ahora puedes saber tu contraseña descirada por si lo llegaste a olvidar.
